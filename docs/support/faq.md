# 자주 묻는 질문

## GGO CE는 원본 ClassicUO와 같은 폴더에 설치해도 되나요?

가능은 하지만 권장하지 않습니다. 별도 폴더에 설치하면 업데이트와 복구가 훨씬 안전합니다.

## 업데이트했는데 버전이 그대로 보입니다

런처와 클라이언트를 모두 종료한 뒤 다시 실행해 보세요. 그래도 같다면 `cuo.dll` 파일 버전 기준으로 확인이 필요합니다.

## 스크립트 설정은 어디에 저장되나요?

대부분 아래 경로에 저장됩니다.

```text
%APPDATA%\GGO_Settings\
```
