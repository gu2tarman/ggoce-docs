# 제보 작성 가이드

좋은 제보는 문제를 빠르게 재현하게 해 줍니다.

## 버그 제보 템플릿

```text
증상:

재현 순서:
1.
2.
3.

기대 동작:

실제 동작:

GGO CE 버전:
런처 버전:
Windows 버전:

첨부:
- 스크린샷
- 로그
```

## 오토루팅 제보

오토루팅 판정 제보에는 아이템 속성 스크린샷이 중요합니다.

- 놓친 아이템인지, 잘못 주운 아이템인지
- 루팅 모드
- 아이템 전체 속성
- 점수 또는 대시보드 표시

## 업데이트 제보

업데이트 문제는 설치 경로와 오류 메시지가 중요합니다.

- 설치 경로
- 업데이트 중 멈춘 단계
- 백신 또는 보안 프로그램 사용 여부
- 런처 로그 또는 화면 캡처
