# HUD 스탯바

HUD 스탯바는 HP, 마나, 스태미너를 한눈에 볼 수 있는 GGO CE 전용 상태바입니다.

## 원본 CUO와 차이

원본 CUO는 기본 스테이터스 검프와 체력바 중심으로 상태를 확인합니다. GGO CE는 별도 HUD 스탯바를 제공해 전투 중에도 주요 자원을 작고 고정된 형태로 볼 수 있게 했습니다.

## 켜는 위치

- 상단 메뉴바 `HUD`
- 채팅 명령어 `-hudbar`
- `Options → GGO Custom → HUD 스탯바 사용`

## 주요 기능

### HP/마나/스태미너 표시

세 자원을 한 줄에서 볼 수 있습니다. 창을 따로 열어두지 않아도 현재 상태를 계속 추적할 수 있습니다.

### 위치 저장

HUD 스탯바 위치와 열림 상태는 프로필에 저장됩니다. 다음 접속 때도 같은 위치로 복원됩니다.

### 회복 헬퍼

HUD 스탯바를 더블클릭하면 회복 헬퍼가 동작합니다. 옵션에서 `Off`, `Magery`, `Chivalry` 중 사용할 학파를 선택할 수 있습니다. 독 상태에서는 해독을 우선 시도합니다.

### 버프바와 접합

향상된 버프 바를 사용 중이면 HUD 스탯바와 붙여서 함께 움직일 수 있습니다. 여러 상태 UI를 한 덩어리로 관리하고 싶을 때 유용합니다.

### 툴팁 확장

HUD 스탯바에 마우스를 올리면 [스탯 인사이트와 스탯 가드](stat-insight-guard-lmc.md) 정보가 함께 표시됩니다.

## 주의 사항

- HUD 스탯바가 중복 생성된 기록이 있어도 최신 버전에서는 하나만 남도록 정리합니다.
- 회복 헬퍼는 현재 타겟/주문 상태와 충돌할 수 있으므로 전투 중 사용 흐름을 먼저 확인하는 것이 좋습니다.
