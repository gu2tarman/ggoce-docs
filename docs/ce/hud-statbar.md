# HUD 스탯바

HP, 마나, 스태미너를 한 줄로 보는 GGO CE 전용 상태바입니다.

## 켜는 위치

- 상단 메뉴바 `HUD`
- 채팅 명령어 `-hudbar`
- `Options → GGO Custom → HUD 스탯바 사용`

## 기능

- 위치와 열림 상태를 저장합니다.
- 더블클릭하면 회복 헬퍼가 동작합니다.
- 회복 헬퍼 학파는 `Off`, `Magery`, `Chivalry` 중 선택합니다.
- 독 상태에서는 해독을 우선 시도합니다.
- 향상된 버프 바와 붙여서 함께 이동할 수 있습니다.
- 마우스를 올리면 [스탯 인사이트](stat-insight-guard-lmc.md) 정보가 표시됩니다.

## 주의 사항

- HUD 스탯바가 중복 생성된 기록은 최신 버전에서 하나만 남도록 정리됩니다.
- 회복 헬퍼는 주문/타겟 상태와 충돌할 수 있으니 먼저 사용 흐름을 확인하세요.
