# HUD 스탯바

![image.png](/ggoce-docs/assets/images/image.png)

HP, 마나, 스태미너를 한 줄로 보는 GGO CE 전용 상태바입니다.

## 켜는 위치

- 상단 메뉴바 `HUD`
- 채팅 명령어 `-hudbar`
- `Options → GGO Custom → HUD 스탯바 사용`

## 기능

- 위치와 열림 상태를 저장합니다.
- 왼쪽 마우스 싱글 클릭으로 캐릭터 본체에 클릭해서 뜨는 팝업 메뉴를 그대로 호출합니다.
- 더블클릭하면 회복 헬퍼가 동작합니다. (체력이 닳았으면 힐, 중독이면 해독)
- 회복 헬퍼 학파는 `Off`, `Magery`, `Chivalry` 중 선택합니다.
- 향상된 버프 바와 붙여서 함께 이동할 수 있습니다. (버프 바를 상단에 붙이면 스탯바에 버프가 가려집니다. 상단에 부착하는것만 피해주세요)
- 마우스를 올리면 [스탯 인사이트](stat-insight-guard-lmc.md) (순수 스탯 + 장비 스탯, 스탯 총합, 마나 소모 감소 정보)가 표시됩니다.
