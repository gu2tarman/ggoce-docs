# GGO CE 기능 둘러보기

GGO CE에서 추가되거나 확장된 기능 목록입니다. 원본과의 큰 차이는 [원본 CUO와 차이](compare-original.md)에 따로 정리했습니다.

슬라이드 형태로 빠르게 보고 싶다면 [기능 둘러보기 슬라이드](https://docs.google.com/presentation/d/1oVyYy87_WiF52fgS7ioUDICbH5Qpw4S2QFEPHOPN4B0/present?slide=id.p1)를 확인하세요.

## 기능 목록

| 기능 | 설명 |
| --- | --- |
| [그리드 뷰 확장](grid-view.md) | 일반 컨테이너 그리드, 검색, 스크롤, 접합, 즐겨찾기 등 |
| [오토루팅](auto-loot.md) | 점수 기반 루팅, 커스텀 목록, 장비 추천, 기록 공유 |
| [HUD 스탯바](hud-statbar.md) | 모던 스킨, 회복 헬퍼, 스탯 툴팁, 클릭 액션 지원 |
| [체력바와 발밑 헬스바](healthbars.md) | 모던 스킨, 파티원 발 밑 체력 상시 표시 옵션, 접속 감지 자동 숨김, 클릭 액션 지원 |
| [페이퍼돌과 장비 슬롯](paperdoll-equipment.md) | 가슴/허리띠 슬롯 추가, 장비 내구도 위험 알림 |
| [스탯 인사이트와 스탯 가드](stat-insight-guard-lmc.md) | 순수 스탯 추정, 목표 스탯 자동 잠금, 실제 마감 수치 계산 |
| [향상된 버프 바와 모던 UI](buffbar-modern-ui.md) | 버프 시간 표시, GGO UI 프레임, HUD 스탯바와 접합 지원 |
| [자동 복구와 패스파인딩](reconnect-pathfinding.md) | 프리징 감지, 재접속, 패스파인딩(길찾기) 기능 강화 및 안정성 향상 |
| [렌더러와 표시 설정](renderer-branding.md) | 렌더러/언어 선택, 캐릭터별 창 크기 저장, 일부 TTF 폰트 지원 |
| [애니메이션 캐시 정리](animation-cache.md) | `flushanim` 명령 (장시간 사냥시 그래픽 과부하 크래시 방지) |
| [원본 CUO 패치 반영](upstream-fixes.md) | 원본 CUO 버그 수정과 GGO CE 안정성 보강 |

## 옵션 위치

주요 설정 탭:

- `Options → GGO Custom`
- `Options → GGO Grid View`
- `Options → GGO Auto Loot`

세부 위치는 [옵션 위치](options.md)를 참고하세요.
