# 패치노트

런처 공지는 짧은 요약을 보여주고, 자세한 변경 사항은 이 페이지에 누적합니다.

## GGO CE 1.6.1.4

### 체력바 클릭 액션

- GGO 체력바와 HUD 스탯바를 한 번 클릭하면 대상의 컨텍스트 메뉴가 마우스 위치에 열립니다.
- GGO 체력바를 Alt+클릭하면 해당 대상 따라가기 기능을 시작합니다.
- 타겟 커서가 떠 있을 때는 타겟 지정이 우선됩니다.

### 발밑 헬스바 상시 표시

- 본인, 파티원, 내 펫의 발밑 약식 체력바를 타겟 여부와 무관하게 항상 표시하는 옵션이 추가되었습니다.
- 본인은 불투명, 파티원/펫은 반투명으로 표시되어 실제 타겟과 구분됩니다.

## GGO CE 1.6.1.3

### 스탯 인사이트

- HUD 스탯바에서 순수 스탯과 장비 증가분을 분리해 표시합니다.
- 착용 150 초과분, 순수 125 초과분 추정, 총합 한계를 함께 확인할 수 있습니다.

### 스탯 가드

- 목표 스탯에 도달하면 자동으로 잠금 처리합니다.
- 옵션에서 기능을 끄거나 기준을 조정할 수 있습니다.

### 마나 소모 감소 미터

- 스탯바 툴팁에서 LMC 실효값과 최대값을 확인할 수 있습니다.
- 아이템 합계와 재질 보너스를 분리해 계산합니다.

### 수정 사항

- 캐릭터 전환 시 창 크기가 로그인 화면 크기로 덮어써질 수 있던 문제를 수정했습니다.

## 이전 패치노트 옮기기

기존 `ggoce-deploy/notice.json`의 패치노트를 이 페이지 또는 버전별 파일로 순차 이전하면 됩니다.
