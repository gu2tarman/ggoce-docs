# 패치노트

런처 공지는 짧은 요약을 보여주고, 자세한 변경 사항은 이 페이지에 누적합니다.

이 페이지는 GGO CE 클라이언트 변경 이력입니다. 런처 자체 변경 이력은 각 런처 GitHub Release에서 확인할 수 있습니다.

## GGO CE 1.6.1.4

### 체력바 클릭 액션

- GGO 체력바와 HUD 스탯바를 한 번 클릭하면 대상의 컨텍스트 메뉴가 마우스 위치에 열립니다.
- GGO 체력바를 Alt+클릭하면 해당 대상 따라가기 기능을 시작합니다.
- 타겟 커서가 떠 있을 때는 타겟 지정이 우선됩니다.

### 발밑 헬스바 상시 표시

- 본인, 파티원, 내 펫의 발밑 약식 체력바를 타겟 여부와 무관하게 항상 표시하는 옵션이 추가되었습니다.
- 본인은 불투명, 파티원/펫은 반투명으로 표시되어 실제 타겟과 구분됩니다.
- 몬스터는 기존과 동일하게 타겟 시에만 표시됩니다.

## GGO CE 1.6.1.3

### 스탯 인사이트

- HUD 스탯바에서 순수 스탯과 장비 증가분을 분리해 표시합니다.
- 착용 150 초과분, 순수 125 초과분 추정, 총합 한계를 함께 확인할 수 있습니다.

### 스탯 가드

- 목표 스탯에 도달하면 자동으로 잠금 처리합니다.
- 옵션에서 기능을 끄거나 기준을 조정할 수 있습니다.

### 마나 소모 감소 미터

- 스탯바 툴팁에서 LMC 실효값과 최대값을 확인할 수 있습니다.
- 아이템 합계와 재질 보너스를 분리해 계산합니다.
- 상한을 넘겨 낭비 중인 수치와 아직 올릴 여유가 있는지를 바로 확인할 수 있습니다.

### 수정 사항

- 캐릭터 전환 시 창 크기가 로그인 화면 크기로 덮어써질 수 있던 문제를 수정했습니다.

## GGO CE 1.6.1.2

### 파티/펫 체력바 자동 숨김

- 꺼내둔 아군과 펫 체력바가 접속 상태에 따라 자동으로 숨겨지고 복원됩니다.
- 파티에 있는 동안은 항상 유지되고, 파티 해제나 시야 이탈 후 유예 시간이 지나면 조용히 사라집니다.
- 다시 감지되면 원래 자리와 앵커 상태로 복원됩니다.

### GGO 체력바 스킨

- 아군과 펫 체력바를 이름이 바 안에 들어간 컴팩트 바 형태로 표시합니다.
- 독 상태와 저체력 상태를 색상과 펄스로 더 쉽게 구분할 수 있습니다.
- 더블클릭 회복 헬퍼, MP 줄 표시, 바 높이 조절을 지원합니다.

### 앵커와 조작 개선

- 혼자 보이는 바를 다른 바에 끌어다 놓으면 기존 그룹에서 분리되어 새로 앵커됩니다.
- 우클릭으로 해당 체력바만 앵커에서 분리하고 닫을 수 있습니다.
- 숨겨진 바는 앵커 대상과 ALT 잠금 표시에서 제외됩니다.

## GGO CE 1.6.1.0

### 그리드 뷰 조작 방식 개선

- 일반 컨테이너 그리드 뷰에서 클릭, 더블클릭, 드래그가 실제 아이템 상호작용처럼 동작하도록 개선했습니다.
- 그리드 뷰에 보이는 아이템을 타겟 지정 대상으로 선택할 수 있습니다.
- 일반 컨테이너에서 Shift+클릭은 해당 아이템을 즉시 꺼내는 동작으로 변경되었습니다.

### 시체 그리드 안내

- 시체 그리드는 루팅 보조 목적이므로 기본 조작 방식이 유지됩니다.
- 시체 그리드에서는 일반 클릭으로 루팅하고, Shift+클릭 또는 Shift+더블클릭으로 실제 아이템 조작을 사용할 수 있습니다.

### 수정 사항

- HUD 스탯바가 중복 생성되어 겹쳐 보일 수 있던 문제를 수정했습니다.
- 이미 중복 저장된 상태에서도 다음 실행/복원 시 하나만 남도록 자동 정리됩니다.
- 테스트용 그리드 접합 위치 보정 옵션을 제거했습니다.

## GGO CE 1.6.0.9

### 그리드 접합과 즐겨찾기

- 일반 컨테이너 그리드 뷰끼리 붙여서 함께 이동할 수 있는 접합 기능을 추가했습니다.
- 접합 상태 저장과 복원을 지원합니다.
- Ctrl+클릭 즐겨찾기와 Ctrl+드래그 즐겨찾기 순서 조정을 추가했습니다.
- 시체 그리드는 접합 대상에서 제외됩니다.

### 안내

- 접합된 그리드는 우클릭으로 닫을 수 있습니다.
- 크기 조절을 시작하면 접합이 자동으로 해제됩니다.

## GGO CE 1.6.0.8

### 그리드 뷰 위치 복원 핫픽스

- 그리드 뷰 위치 복원 후 같은 컨테이너를 일반 컨테이너로 열 때 위치가 겹칠 수 있던 문제를 수정했습니다.
- 일반 컨테이너 위치 저장과 그리드 뷰 위치 저장이 서로 덮어쓰지 않도록 분리했습니다.

## GGO CE 1.6.0.7

### 커스텀 HUD 스탯바

- HP, 마나, 스태미너를 한눈에 볼 수 있는 GGO HUD 스탯바를 추가했습니다.
- 상단 메뉴바의 `HUD` 버튼, 옵션의 `GGO Custom → HUD 스탯바 사용`, 채팅 명령어 `-hudbar`로 열 수 있습니다.
- 더블클릭하면 회복 헬퍼가 작동합니다.
- 향상된 버프 바와 붙여서 함께 이동할 수 있고, 위치와 열림 상태가 저장됩니다.

### 그리드 뷰 개선

- 클라이언트 종료 전에 열려 있던 그리드 뷰 위치를 기억하도록 개선했습니다.
- 그리드 크기 저장 기록이 과도하게 쌓이지 않도록 오래된 기록을 정리합니다.
- 아이템 칸과 버튼을 제외한 영역을 더 안정적으로 드래그할 수 있게 했습니다.

## GGO CE 1.6.0.6

### 그리드 뷰 모던 UI

- 그리드 컨테이너와 시체 뷰어에 장식 프레임 패널을 적용했습니다.
- 분류는 슬롯 좌상단, 점수는 우하단에 표시되도록 정리했습니다.

### 크기 조절과 스크롤

- 우하단 모서리 드래그로 그리드 크기를 슬롯 단위로 조절할 수 있습니다.
- 아이템이 창보다 많으면 마우스 휠 또는 우측 스크롤바로 이동합니다.
- 시체 그리드는 공용 크기로, 일반 컨테이너는 개별 크기로 저장됩니다.

### 옵션 개편

- 그리드 관련 옵션을 `GGO Grid View` 탭으로 이동했습니다.
- 슬롯 크기 옵션을 추가해 슬롯과 아이템 아이콘 크기를 `50~80` 범위에서 조절할 수 있습니다.

## GGO CE 1.6.0.5

### 표시 버전 핫픽스

- 로그인 화면의 GGO CE 버전 표기가 실제 빌드 버전을 따라가도록 수정했습니다.
- 게임 창 제목의 GGO CE 버전 표기가 실제 빌드 버전을 따라가도록 수정했습니다.
- 클라이언트 표시 버전과 배포 manifest 버전을 1.6.0.5로 맞췄습니다.

## GGO CE 1.6.0.4

### 오토루팅 장비 추천

- 무기, 방어구, 방패 추천 표시와 일반모드 전방 정렬을 시체 그리드에서만 적용하도록 조정했습니다.
- 일반 컨테이너의 점수모드는 기존처럼 수동 채점 결과를 볼 수 있게 유지했습니다.
- 추천 태그를 더 작게 정리했습니다.

### 시체 자동열기와 루팅 간격

- 검프가 실제로 열렸을 때만 자동오픈 완료로 등록해, 거리 문제로 열리지 않은 시체는 다음 기회에 다시 시도합니다.
- 닫혔거나 이미 완료된 시체가 큐에 남아 뒤늦게 거리 취소 메시지를 띄우는 상황을 방어했습니다.
- 오토룻 설정에 루팅 시간 조정값을 추가했습니다.

## GGO CE 1.6.0.3

### 오토루팅 커스텀 목록

- 커스텀 아이템마다 이름 일치 토글을 추가했습니다.
- 이름 일치를 켜면 같은 베이스라도 지정한 이름과 일치하는 아이템만 줍습니다.

### 원본 CUO 패치 일부 반영

- 커스텀 하우징 단계에서 발생할 수 있던 버그를 일부 수정했습니다.
- 마스터리 스펠북에서 현재 활성화된 마스터리 어빌리티가 정상 표시되도록 수정했습니다.
- 거래창 제안 한도가 골드/플래티넘 합산 잔액 기준으로 계산되도록 개선했습니다.
- 책 페이지 줄 수와 글자색을 원본 클라이언트와 더 일치하도록 개선했습니다.
- 맵 청크 관련 크래시와 렌더 오류를 수정했습니다.

## GGO CE 1.6.0.2

### 렌더러 선택

- DirectX 11 렌더러 선택이 실제로 적용되지 않던 문제를 수정했습니다.
- `Options → GGO Custom` 탭에서 그래픽 렌더러를 직접 선택할 수 있도록 추가했습니다.
- 렌더러 변경은 클라이언트 재시작 후 적용됩니다.

## GGO CE 1.6.0.1

### 무기와 방어구 판정

- 기준을 통과한 무기, 방어구, 방패를 알려 주는 판정 기능을 추가했습니다.
- 방어구와 방패는 스탯과 주요 옵션 점수를 합산해 판정합니다.
- 무기는 체력 흡수와 마나 흡수가 모두 기준치 이상일 때 판정합니다.
- 판정된 장비는 머리 위 알림과 시체 그리드 표시로 직접 확인할 수 있습니다.

### 옵션

- `Options → GGO Auto Loot → 무기 및 방어구 점수 판정`에서 무기 판정과 방어구/방패 판정을 각각 켜고 끌 수 있습니다.

## GGO CE 1.6.0.0

### 모던 UI 1차 적용

- 오토루팅 요약 검프와 상세 검프의 배경 및 버튼 시인성을 개선했습니다.
- 페이퍼돌 하단 이름표를 새 디자인으로 교체했습니다.
- 향상된 버프 바의 고정 기준 바 디자인을 적용했습니다.
- `GGO 모던 UI 사용` 옵션이 기본 켜짐으로 적용됩니다.

## GGO CE 1.5.0.6

### 향상된 버프 바

- 활성 버프마다 남은 시간 막대와 카운트다운을 표시하는 향상된 버프 바를 추가했습니다.
- 캐릭터별 설정과 막대 색상 조정을 지원합니다.

### 그리드뷰 개선

- 그리드뷰어에 보이는 컨테이너 아이템을 Shift+클릭하면 컨테이너를 열 수 있습니다.
- 그리드뷰 영역 또는 그리드뷰 안의 컨테이너 아이콘에 아이템을 드래그 드랍해 이동할 수 있습니다.

### 버프 툴팁 수정

- 버프 툴팁에 정렬 문자열이 잘못 출력되던 오류를 수정했습니다.
- 한글 툴팁에서 일부 버프 수치가 누락되던 문제를 수정했습니다.

## GGO CE 1.5.0.5

### 애니메이션 캐시 정리

- 채팅 명령어 `flushanim`으로 캐릭터, 몬스터, 장비 애니메이션 캐시를 비울 수 있습니다.
- 장시간 다중 클라이언트 환경에서 누적되는 그래픽 메모리를 클라이언트 재시작 없이 정리할 수 있습니다.

### 사용 시 주의

- 명령 실행 직후 화면의 대상이 다시 로드되어 잠시 GPU 사용량이 늘 수 있습니다.
- 여러 클라이언트에서 동시에 호출하지 않는 것을 권장합니다.

## GGO CE 1.5.0.4

### 자동 복구 시스템

- 서버 응답이 멈추는 프리징을 감지하면 자동 재접속을 시도합니다.
- 서버 점검/리붓 중에는 서버가 살아날 때까지 대기했다가 다시 접속합니다.
- 로그인 정체, 연결 끊김 팝업, 반복 실패 상황을 복구 흐름에 포함했습니다.
- 복구 과정을 Discord 웹훅 알림으로 받을 수 있습니다.

### 패스파인딩 개선

- 목적지가 막혀 있어도 갈 수 있는 곳까지 이동하도록 개선했습니다.
- 경로 계산 과부하로 클라이언트가 멈추지 않도록 보호 장치를 추가했습니다.

## GGO CE 1.5.0.3

### 오토 룻 요약 검프

- 요약 검프에서 오토루팅과 시체 자동 열기 옵션을 바로 켜고 끌 수 있게 변경했습니다.

### 오토루팅 오류 수정

- 인간계열 몬스터 시체에서 오토루팅이 작동하지 않을 수 있던 오류를 수정했습니다.
- 실제 아이템 이동 실패가 아닌 상황을 아이템 이동불가로 오판정할 수 있던 문제를 수정했습니다.

## GGO CE 1.5.0.2

### 오토 룻 요약 검프

- 오른쪽 메뉴를 상세, 추가, 커스텀 구성으로 정리했습니다.
- 상세 메뉴에서 파밍 기록 상세 창을 열 수 있습니다.
- 추가 메뉴에서 아이템을 선택해 커스텀 루팅 목록에 바로 등록할 수 있습니다.
- 커스텀 메뉴에서 기존 커스텀 목록 편집 창을 바로 열 수 있습니다.
- 최고점 표기를 `Max :` 형식으로 조정했습니다.

## GGO CE 1.5.0.1

### 오토 룻 요약 검프

- 오토루팅 옵션을 꺼둔 상태에서 다시 켜면 요약 검프가 자동으로 열립니다.
- 요약 검프 위치와 열림 상태가 저장됩니다.
- 상단 메뉴바의 `Auto Loot` 버튼으로 요약 검프를 직접 열 수 있습니다.

### 파밍 기록
