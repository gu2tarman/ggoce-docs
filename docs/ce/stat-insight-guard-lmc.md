# 스탯 인사이트와 스탯 가드

스탯 인사이트와 스탯 가드는 장비를 착용한 상태에서도 순수 스탯과 장비 증가분을 추정하고, 목표에 도달한 스탯을 자동으로 잠그는 기능입니다.

## 원본 CUO와 차이

원본 CUO에서는 화면에 보이는 STR/DEX/INT가 장비와 버프가 반영된 결과인지 사용자가 직접 판단해야 합니다. GGO CE는 장비 OPL을 읽어 순수 스탯과 장비 보너스를 분리해 보여주고, 목표값에 도달하면 잠금까지 보조합니다.

## 켜는 위치

`Options → GGO Custom`

## 스탯 인사이트

HUD 스탯바에 마우스를 올리면 다음 정보를 보여줍니다.

- 표시 스탯
- 장비 증가분
- 추정 순수 스탯
- 착용 기준 초과분
- 순수 총합과 총합 한계

순수치는 표시치가 착용 상한보다 낮은 순간 자동 측정되어 캐릭터별로 저장됩니다. 표시치가 150에 고정된 스탯은 장비를 잠시 벗으면 측정이 쉬워집니다.

## 스탯 가드

스탯이 목표에 도달하면 자동으로 잠금 상태로 바꿉니다.

기본 목표:

| 항목 | 기본값 |
| --- | --- |
| STR 순수 목표 | 125 |
| DEX 순수 목표 | 125 |
| INT 순수 목표 | 125 |
| 착용 기준 | 150 |

스탯 버프나 디버프가 걸린 상태에서는 판정을 쉬어 잘못 잠그는 상황을 줄입니다. 사용자가 수동으로 잠금을 되돌리면 그 의도를 우선합니다.

## 숨은 초과분 조정

표시는 150에 고정되어 있지만 실제 순수치가 더 높을 수 있습니다. 내림 조정 중에는 초과분이 정확히 0이 되는 순간 잠금해, 표시 150을 유지하면서 낭비를 줄이는 것이 목표입니다.

## LMC 미터

HUD 스탯바 툴팁에 LMC 실효값과 최대값이 표시됩니다.

계산 기준:

- 아이템 LMC 합계는 기본 상한 `40`을 적용합니다.
- 재질 보너스는 별도로 더합니다.
- 스터디드/본/스톤 계열은 조각당 `+3`으로 계산합니다.
- 금속/드래곤 계열은 조각당 `+1`로 계산합니다.
- 최대 5조각까지 반영합니다.
- 메이지 아머가 붙은 장비는 재질 보너스가 무효로 처리됩니다.

툴팁은 아이템 LMC가 낭비 중인지, 아직 올릴 여유가 있는지도 함께 보여줍니다.

## 제보 포인트

- 특정 장비의 스탯 보너스가 반영되지 않는 경우
- 순수 스탯 추정이 실제와 다르게 보이는 경우
- LMC 재질 보너스가 잘못 계산되는 경우
- 스탯 버프 중 자동 잠금이 예상과 다르게 동작하는 경우
