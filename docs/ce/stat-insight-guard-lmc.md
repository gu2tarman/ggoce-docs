# 스탯 인사이트와 스탯 가드

장비를 착용한 상태에서 순수 스탯과 장비 증가분을 분리해 보고, 목표에 도달한 스탯을 자동으로 잠그는 기능입니다.

## 켜는 위치

`Options → GGO Custom`

## 스탯 인사이트

HUD 스탯바에 마우스를 올리면 다음 값을 봅니다.

- 표시 스탯
- 장비 증가분
- 추정 순수 스탯
- 착용 기준 초과분
- 순수 총합과 총합 한계

순수치는 표시치가 착용 상한보다 낮은 순간 자동 측정되어 캐릭터별로 저장됩니다. 표시치가 150에 고정된 스탯은 장비를 잠시 벗으면 측정이 쉬워집니다.

## 스탯 가드

목표에 도달한 스탯을 자동으로 잠급니다.

| 항목 | 기본값 |
| --- | --- |
| STR 순수 목표 | 125 |
| DEX 순수 목표 | 125 |
| INT 순수 목표 | 125 |
| 착용 기준 | 150 |

스탯 버프/디버프 중에는 판정을 쉬고, 사용자가 수동으로 잠금을 되돌리면 그 상태를 우선합니다.

## LMC 미터

HUD 스탯바 툴팁에 LMC 실효값과 최대값을 표시합니다.

- 아이템 LMC 합계는 기본 상한 `40`을 적용합니다.
- 재질 보너스는 별도로 더합니다.
- 스터디드/본/스톤 계열은 조각당 `+3`으로 계산합니다.
- 금속/드래곤 계열은 조각당 `+1`로 계산합니다.
- 최대 5조각까지 반영합니다.
- 메이지 아머가 붙은 장비는 재질 보너스가 무효로 처리됩니다.

## 제보 포인트

- 특정 장비의 스탯 보너스가 반영되지 않는 경우
- 순수 스탯 추정이 실제와 다르게 보이는 경우
- LMC 재질 보너스가 잘못 계산되는 경우
- 스탯 버프 중 자동 잠금이 예상과 다르게 동작하는 경우
