# 스탯 인사이트와 스탯 가드

장비의 옵션을 분석하여 캐릭터의 순수 스탯과 장비 증가분을 분리하여 표시합니다. 

옵션에서 설정한 목표에 도달하면 스탯을 자동으로 잠궈, 낭비되는 스탯이 없도록 도와줍니다.

스탯이 올라가는 버프 등에 영향을 받은 경우 측정 값이 틀어질 수 있습니다. 실 스탯 체크시에는 스탯 변경이 되는 외부 요인에 영향을 받지 않도록 해주세요. (체크해서 최적 수치를 조정해 둔 이후에는 보실 일이 없으시겠지만 이후에 영향 받은 버프에도 측정값이 오염될 수 있습니다.)

![image.png](/ggoce-docs/assets/images/image-9.png)

## 켜는 위치

`Options → GGO Custom`

## 스탯 인사이트

HUD 스탯바에 마우스를 올리면 다음 값을 봅니다.

- 표시 스탯 (상세 스테이터스 바에 표시되고 있는 수치)
- 장비 증가분 (장비 옵션으로 획득한 수치)
- 추정 순수 스탯 (장비 증가분, 스탯 총합 등 역산을 통해 추정)
- 착용 기준 초과분 (낭비되고 있는 것으로 예상되는 스탯)
- 순수 총합과 총합 한계

순수치는 표시치가 착용 상한보다 낮은 순간 자동 측정되어 캐릭터별로 저장됩니다. 최초 체크시 이미 150을 달성한 스탯은 해당 스탯을 올려주는 장비를 잠시 벗으면 측정이 쉬워집니다.

## 스탯 가드

목표에 도달한 스탯을 자동으로 잠급니다. (수치 변경 가능)


| 항목 | 기본값 |
| --------- | --- |
| STR 순수 목표 | 125 |
| DEX 순수 목표 | 125 |
| INT 순수 목표 | 125 |
| 착용 기준 | 150 |


스탯 버프/디버프 중에는 판정을 쉬고, 사용자가 수동으로 잠금을 되돌리면 그 상태를 우선합니다.

(ex : 자동 잠금 -> 수동 잠금 해제 -> 잠금 해제 상태 유지)

## LMC (마나 소모 감소) 미터

HUD 스탯바 툴팁에 마나 소모 감소의 실효값과 최대값을 표시합니다.

- 아이템 LMC 합계는 기본 상한 `40`
- 착용 아이템을 체크하여 재질 보너스를 별도로 계산합니다. (스텃/본/스톤 = +3, 금속/드래곤 = +1)
- 게임에서 보여주지 않던 내 진짜 최대 수치와 현재 수치를 계산하여 얼마나 부족한지, 얼마나 넘치는지 보여줍니다.

## 제보 포인트

- 특정 장비의 스탯 보너스가 반영되지 않는 경우
- 순수 스탯 추정이 실제와 다르게 보이는 경우
- LMC 재질 보너스가 잘못 계산되는 경우
- 스탯 버프 중 자동 잠금이 예상과 다르게 동작하는 경우
