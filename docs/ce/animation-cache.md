# 애니메이션 캐시 정리

`flushanim`은 장시간 다중 클라이언트 환경에서 누적된 애니메이션 캐시를 수동으로 정리하는 GGO CE 명령입니다.

## 원본 CUO와 차이

원본 CUO에서는 일반 사용자가 플레이 중 애니메이션 캐시를 직접 정리하는 흐름이 없습니다. GGO CE는 장시간 사냥과 다중 클라이언트 운영을 고려해 수동 정리 명령을 추가했습니다.

## 사용 방법

채팅 입력창에서 명령 입력을 연 뒤 다음 명령을 입력합니다.

```text
flushanim
```

또는 명령 prefix를 포함해 사용하는 환경에서는 다음처럼 입력합니다.

```text
-flushanim
```

## 언제 쓰나요?

- 여러 클라이언트를 오래 켜 둔 경우
- 장시간 매크로 또는 사냥 후 메모리 사용량이 부담되는 경우
- 클라이언트 재시작 없이 애니메이션 캐시를 비우고 싶은 경우

## 주의 사항

- 실행 직후 화면의 대상 애니메이션이 다시 로드됩니다.
- 잠시 그래픽카드 사용량이 늘 수 있습니다.
- 여러 클라이언트에서 동시에 실행하지 말고 시간차를 두세요.
- 일반적인 플레이에서는 자주 사용할 필요가 없습니다.
