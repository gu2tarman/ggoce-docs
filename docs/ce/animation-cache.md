# 애니메이션 캐시 정리

`flushanim`은 애니메이션 캐시를 수동으로 비우는 명령입니다. 장시간 다중 클라이언트 환경에서만 필요합니다.

## 사용 방법

명령 입력창에 입력합니다.

```text
flushanim
```

환경에 따라 prefix를 포함해 입력합니다.

```text
-flushanim
```

## 사용 상황

- 여러 클라이언트를 오래 켜 둔 경우
- 장시간 사냥 후 메모리 사용량이 부담되는 경우
- 클라이언트 재시작 없이 캐시를 비우고 싶은 경우

## 주의 사항

- 실행 직후 화면의 대상 애니메이션이 다시 로드됩니다.
- 잠시 그래픽카드 사용량이 늘 수 있습니다.
- 여러 클라이언트에서 동시에 실행하지 마세요.
