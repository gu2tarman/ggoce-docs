# 렌더러와 표시 설정

그래픽 렌더러, 클라이언트 언어, 창 크기, 일부 TTF 폰트 표시를 옵션에서 관리할 수 있습니다.

## 그래픽 렌더러 선택

`Options → GGO Custom → 그래픽 렌더러`

- Auto
- OpenGL
- Vulkan
- DirectX 11

렌더러 변경은 클라이언트 재시작 후 적용됩니다.

## 클라 언어 선택

옵션에서 클라이언트 언어를 선택할 수 있습니다. 클라이언트 재시작 후 적용됩니다.

`Options → GGO Custom → 언어`

## 게임 창 크기

GGO CE는 게임 창 최소 크기 제한을 제거해, 플레이 환경에 맞춰 창을 더 자유롭게 줄이거나 키울 수 있습니다.

창 크기는 캐릭터별 프로필에 저장됩니다. 사냥 캐릭터, 생산 캐릭터, 보조 캐릭터처럼 서로 다른 화면 배치를 쓰는 경우에도 각 캐릭터의 마지막 창 크기와 위치를 따로 기억합니다.

관련 위치:

- `Options → Video → Game Window Size`
- 게임 화면 우측 하단 크기 조절 핸들

## 커스텀 TTF 폰트 일부 지원

툴팁과 모던 UI 일부 영역에서 TTF 폰트를 지원합니다. 기본 비트맵 폰트보다 한글 가독성이 필요한 곳을 우선 적용한 기능입니다.

관련 위치:

- `Options → Tooltip → TTF 폰트 사용 [아이템 툴팁]`
- `Options → Tooltip → TTF 폰트 크기`
- `Options → Tooltip → TTF 폰트 선택`

!!! note
    모든 게임 텍스트가 TTF로 바뀌는 것은 아닙니다. 현재는 툴팁, 페이퍼돌 이름표, GGO 모던 UI 일부처럼 GGO CE에서 직접 그리는 영역을 중심으로 지원합니다.
