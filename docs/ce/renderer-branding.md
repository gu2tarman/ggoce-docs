# 렌더러와 표시 버전

GGO CE는 그래픽 렌더러 선택과 배포판 식별을 더 명확하게 하기 위한 기능을 포함합니다.

## 그래픽 렌더러 선택

### 원본 CUO와 차이

원본 환경에서는 렌더러 선택이 사용자의 설정 파일이나 실행 환경에 의존하는 경우가 많습니다. GGO CE는 옵션에서 렌더러를 직접 선택할 수 있도록 했습니다.

### 켜는 위치

`Options → GGO Custom → 그래픽 렌더러`

선택지:

- Auto
- OpenGL
- Vulkan
- DirectX 11

렌더러 변경은 클라이언트 재시작 후 적용됩니다.

## GGO CE 표시 버전

GGO CE는 창 제목과 로그인 화면 하단에 GGO CE 버전을 표시합니다.

예시:

```text
CUO GGO CE - 1.6.1.4
```

이 표시는 사용자가 원본 ClassicUO와 GGO CE를 헷갈리지 않도록 돕고, 제보 시 실제 실행 중인 배포판을 확인하는 기준이 됩니다.

## 업데이트 감지 기준

런처가 GGO CE 버전을 판단할 때는 `ClassicUO.exe`가 아니라 `cuo.dll`의 파일 버전을 기준으로 봅니다. 그래서 클라이언트 코드 변경만 있는 hotfix는 `cuo.dll`과 `version.txt`만 배포될 수 있습니다.

## 제보 포인트

- 옵션에서 DirectX 11을 골랐는데 재시작 후 적용되지 않는 경우
- 창 제목 또는 로그인 화면 버전이 런처 표시와 다른 경우
- 원본 ClassicUO 폴더와 GGO CE 폴더가 섞인 것으로 의심되는 경우
