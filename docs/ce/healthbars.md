# 체력바와 발밑 헬스바

파티원과 펫 상태를 보기 쉽게 만드는 체력바 기능입니다.

## 켜는 위치

`Options → GGO Custom`

## 기능

- GGO 체력바 스킨: 이름이 바 안에 들어간 컴팩트 체력바입니다.
- MP 줄 표시: 본인/파티원 체력바 아래에 MP 줄을 표시합니다.
- 높이 조절: `16~32` 범위, 기본값 `27`입니다.
- 자동 숨김: 파티 해제, 접속 종료, 시야 이탈 후 체력바를 숨기고 다시 감지되면 복원합니다.
- 발밑 헬스바: 본인/파티원/내 펫의 약식 체력바를 상시 표시합니다.

## 조작

| 조작 | 동작 |
| --- | --- |
| 한 번 클릭 | 대상 컨텍스트 메뉴를 마우스 위치에 엽니다. |
| Alt+클릭 | 해당 대상을 따라갑니다. |
| 더블클릭 | 회복 헬퍼를 실행합니다. |
| 우클릭 | 해당 체력바를 앵커에서 분리하고 닫습니다. |

타겟 커서가 떠 있으면 타겟 지정이 우선됩니다.

## 주의 사항

- 펫 더블클릭은 기본 탑승 동작과 충돌할 수 있어 회복 헬퍼 대상에서 제외됩니다.
- 본인 체력바 더블클릭은 상세 스테이터스 동작을 유지합니다.
- 발밑 헬스바에서 몬스터는 기존처럼 타겟 시에만 표시됩니다.
