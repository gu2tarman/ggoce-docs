# 체력바와 발밑 헬스바

파티 체력바를 컴팩트하게 바꿔주고, 더블클릭 회복 헬퍼, 체력바 클릭으로 캐릭터를 직접 클릭한 것같은 액션을 지원합니다.  
(싱글클릭 -> 팝업메뉴, 알트 클릭 -> 팔로우 등) 

우호적 대상이 시야에서 사라지면 자동으로 화면에서 감춰 화면을 깨끗하게 유지합니다.

파티원인 경우에는 파티에서 이탈하기 전까지 화면에서 사라져도 체력바를 유지합니다.

![image.png](/ggoce-docs/assets/images/image-12.png)

발 밑 약식 체력바를 파티원과 펫에서 상시 활성화하여 파티원의 체력을 보다 직관적으로 파악할 수 있도록 했습니다.

![image.png](/ggoce-docs/assets/images/image-13.png)

## 켜는 위치

`Options → GGO Custom`

## 기능

- GGO 체력바 스킨: 이름이 바 안에 들어간 컴팩트 체력바입니다.
- MP 줄 표시: 본인/파티원 체력바 아래에 MP 줄을 표시합니다. 컴팩트한 관리를 원하는분은 꺼주세요.
- 높이 조절: `16~32` 범위, 기본값 `27`입니다.
- 자동 숨김: 파티 해제, 접속 종료, 시야 이탈 후 체력바를 숨기고 다시 감지되면 복원합니다.
- 발밑 헬스바: 본인/파티원/내 펫의 약식 체력바를 상시 표시합니다.

## 조작


| 조작 | 동작 |
| ------ | ------------------------ |
| 한 번 클릭 | 대상 컨텍스트 메뉴를 마우스 위치에 엽니다. |
| Alt+클릭 | 해당 대상을 따라갑니다. |
| 더블클릭 | 회복 헬퍼를 실행합니다. |
| 우클릭 | 해당 체력바를 앵커에서 분리하고 닫습니다. |


타겟 커서가 떠 있으면 타겟 지정이 우선됩니다.

## 주의 사항

- 펫 더블클릭은 기본 탑승 동작과 충돌할 수 있어 회복 헬퍼 대상에서 제외됩니다.
- 본인 체력바 더블클릭은 상세 스테이터스 동작을 유지합니다.
- 발밑 헬스바에서 몬스터는 기존처럼 타겟 시에만 표시됩니다.
