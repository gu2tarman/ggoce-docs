# 체력바와 발밑 헬스바

GGO CE는 파티원과 펫을 더 쉽게 관리할 수 있도록 체력바 스킨, 자동 숨김, 클릭 액션, 발밑 헬스바 상시 표시를 추가했습니다.

## 원본 CUO와 차이

| 항목 | 원본 CUO | GGO CE |
| --- | --- | --- |
| 체력바 외형 | 기본 체력바 | 이름이 바 안에 들어간 컴팩트 GGO 스킨 |
| 파티/펫 관리 | 사용자가 직접 열고 닫음 | 접속/시야 상태에 따라 자동 숨김/복원 |
| 조작 | 기본 더블클릭/타겟 중심 | 클릭 메뉴, Alt+클릭 팔로우, 더블클릭 회복 헬퍼 |
| 발밑 표시 | 타겟 중심 | 본인/파티원/내 펫 상시 표시 옵션 |

## 켜는 위치

`Options → GGO Custom`

## GGO 체력바 스킨

아군과 펫 체력바를 GGO 전용 컴팩트 바 형태로 바꿉니다. 이름이 바 안에 들어가고, 독 상태와 저체력 상태가 더 눈에 띄게 표현됩니다.

### MP 줄 표시

본인과 파티원은 HP 아래에 MP 줄을 추가로 표시할 수 있습니다. 체력바를 작게 모아두고 전투 자원을 같이 보고 싶을 때 유용합니다.

### 높이 조절

체력바 높이는 `16~32` 범위에서 조정할 수 있습니다. 기본값은 `27`입니다.

## 파티/펫 체력바 자동 숨김

꺼내둔 아군/펫 체력바를 접속 상태와 시야 상태에 따라 자동으로 숨기거나 복원합니다.

- 파티에 있는 동안은 유지됩니다.
- 파티 해제, 접속 종료, 시야 이탈 후 유예 시간이 지나면 숨깁니다.
- 다시 감지되면 원래 자리와 앵커로 복원합니다.

## 클릭 액션

| 조작 | 동작 |
| --- | --- |
| 한 번 클릭 | 대상 컨텍스트 메뉴를 마우스 위치에 엽니다. |
| Alt+클릭 | 해당 대상을 따라갑니다. |
| 더블클릭 | 회복 헬퍼를 실행합니다. |
| 우클릭 | 해당 체력바를 앵커에서 분리하고 닫습니다. |

타겟 커서가 떠 있을 때는 타겟 지정이 우선됩니다. 더블클릭 회복 헬퍼가 동작하는 상황에서는 클릭 메뉴가 열리지 않습니다.

## 발밑 헬스바 상시 표시

본인, 파티원, 내 펫의 발밑 약식 체력바를 타겟 여부와 무관하게 항상 표시할 수 있습니다.

- 본인은 불투명하게 표시됩니다.
- 파티원과 펫은 반투명하게 표시됩니다.
- 몬스터는 기존처럼 타겟 시에만 표시됩니다.

## 주의 사항

- 펫 더블클릭은 기본 탑승 동작과 충돌할 수 있어 회복 헬퍼 대상에서 제외됩니다.
- 본인 체력바 더블클릭은 상세 스테이터스 동작을 유지합니다.
- 자동 숨김은 파티/시야 정보에 의존하므로 서버 응답이 늦으면 약간 늦게 반영될 수 있습니다.
