# 페이퍼돌과 장비 슬롯

GGO CE는 원본 ClassicUO의 페이퍼돌을 보강해 장비 확인과 관리가 더 쉽도록 했습니다.

![image.png](/ggoce-docs/assets/images/image-17.png)

## 추가 장비 슬롯

원본 페이퍼돌에는 바로 보이지 않던 장비 위치 중 `가슴(Torso)`과 `허리띠(Waist)` 슬롯을 추가했습니다.

- 가슴 슬롯은 페이퍼돌 왼쪽 상단에 표시됩니다.
- 허리띠 슬롯은 페이퍼돌 오른쪽 상단에 표시됩니다.
- 두 슬롯 모두 기존 장비 슬롯처럼 아이템 착용, 해제, 드래그 조작을 지원합니다.

## 장비 내구도 위험 알림

장비 툴팁에서 내구도 정보를 읽어 각 장비 슬롯 테두리 색으로 위험도를 표시합니다. 내구도는 `현재 내구도 / 최대 내구도` 비율로 계산합니다.


| 내구도 비율 | 슬롯 테두리 | 코드 기준 |
| --------------- | ------ | ------------ |
| 50% 이상 | 기본색 | `Hue = 0` |
| 35% 이상 ~ 50% 미만 | 노란색 계열 | `Hue = 1174` |
| 20% 이상 ~ 35% 미만 | 주황색 계열 | `Hue = 1259` |
| 20% 미만 | 빨간색 계열 | `Hue = 33` |


내구도 정보는 아이템 속성 툴팁을 통해 갱신됩니다. 슬롯을 열었을 때 아직 내구도 정보가 없으면 기본색으로 표시되고, 툴팁 정보가 도착한 뒤 위험 색상이 반영됩니다.

## 참고

- 내구도 표시가 없는 아이템은 경고 색상이 적용되지 않습니다.
- 장비를 교체했는데 색상이 늦게 바뀌면 해당 장비에 마우스를 올려 툴팁 정보를 갱신해 주세요.
