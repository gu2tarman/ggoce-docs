# 원본 CUO와 차이

GGO CE는 원본 ClassicUO를 기반으로 한 GGO용 배포판입니다. 기본 조작은 원본을 따르고, 루팅/상태 표시/장시간 플레이 보조 기능을 추가했습니다.

## 차이 요약

| 영역 | 원본 CUO | GGO CE |
| --- | --- | --- |
| 컨테이너 | 기본 컨테이너, Grid Loot | 일반 컨테이너 그리드, 검색, 접합, 즐겨찾기 |
| 루팅 | 수동 루팅 중심 | 점수 기반 오토루팅, 커스텀 목록, 장비 추천 |
| 상태 표시 | 기본 상태창 | HUD 스탯바, 순수 스탯 추정, LMC 실효값 |
| 스탯 관리 | 수동 잠금 | 목표 도달 시 자동 잠금 |
| 체력바 | 기본 체력바 | GGO 스킨, MP 줄, 자동 숨김, 클릭 액션 |
| 버프 표시 | 기본 버프 바 | 남은 시간 막대와 카운트다운이 있는 향상된 버프 바 |
| 장시간 플레이 | 수동 복구 | 프리징 감지, 재접속, 웹훅 알림 |
| 기타 | 기본 캐시/렌더러 | `flushanim`, 렌더러 선택, GGO CE 버전 표시 |

자세한 사용법은 [기능 둘러보기](features.md)에서 항목별로 확인하세요.
