# 원본 CUO와 차이

이 문서는 원본 ClassicUO 기준으로 GGO CE에서 추가되거나 확장된 기능을 한눈에 비교하기 위한 페이지입니다.

## 비교표

| 영역 | 원본 CUO | GGO CE |
| --- | --- | --- |
| 컨테이너 UI | 기본 컨테이너와 Grid Loot 중심 | 일반 컨테이너까지 그리드 뷰 확장, 검색, 스크롤, 크기 저장, 접합, 즐겨찾기 |
| 루팅 | 수동 루팅 또는 기본 Grid Loot | 점수 기반 오토루팅, 커스텀 목록, 장비 추천, 루팅 기록 |
| 상태 표시 | 기본 스탯/상태창 | HUD 스탯바, 순수 스탯 추정, 장비 증가분 분리, LMC 실효값 표시 |
| 스탯 관리 | 사용자가 직접 잠금 관리 | 목표 순수 스탯/착용 스탯 도달 시 자동 잠금 |
| 체력바 | 기본 체력바 | GGO 체력바 스킨, MP 줄, 자동 숨김/복원, 클릭 액션, 발밑 헬스바 상시 표시 |
| 버프 표시 | 기본 버프 바 | 남은 시간 막대와 카운트다운이 있는 향상된 버프 바 |
| 장시간 플레이 | 수동 재접속/재시작 | 프리징 감지, 자동 재접속, 점검 대기, 웹훅 알림 |
| 성능/안정성 보조 | 기본 캐시 관리 | `flushanim` 애니메이션 캐시 정리 명령 |
| 렌더러 | 환경 설정에 의존 | 옵션에서 Auto/OpenGL/Vulkan/DirectX 11 선택 |
| 배포 식별 | ClassicUO 버전 표시 | 로그인 화면/창 제목에 GGO CE 버전 표시 |

## 설명 포인트

GGO CE는 원본 CUO를 대체하는 완전히 다른 클라이언트라기보다, 원본 CUO를 기반으로 GGO 플레이에 필요한 편의 기능을 얹은 배포판입니다. 따라서 기본 조작과 호환성은 원본 CUO를 따르되, 루팅/상태 표시/장시간 플레이/다중 클라이언트 운영에서 반복되는 불편을 줄이는 쪽에 초점이 있습니다.
