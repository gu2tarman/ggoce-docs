# 원본 CUO와 차이

GGO CE는 원본 ClassicUO를 기반으로 필요하다 생각한 기능을 추가하고 강화한 커스텀 클라이언트 입니다.

## 차이 요약

| 영역 | 원본 CUO | GGO CE |
| --- | --- | --- |
| 컨테이너 | 기본 컨테이너, Grid Loot | 일반 컨테이너 그리드, 검색, 접합, 즐겨찾기 |
| 유저 벤더 | 일반 상점 검프 중심 조작 | 그리드 뷰 아이템 클릭으로 구매 버튼 활성화 |
| 루팅 | 수동 루팅 중심 | 점수 기반 오토루팅, 커스텀 목록, 장비 추천 |
| 상태 표시 | 기본 상태창 | HUD 스탯바, 순수 스탯 추정, LMC 실효값 |
| 스탯 관리 | 수동 잠금 | 목표 설정, 목표 도달 시 자동 잠금, 순수 스탯 계산 |
| 체력바 | 기본 체력바 | 커스텀 스킨, 접속 감지 자동 숨김, 클릭 액션 지원 등 |
| 페이퍼돌 | 기본 장비 슬롯 | 가슴과 허리띠 슬롯 추가, 장비 내구도 위험 알림 |
| 버프 표시 | 기본 버프 바 | 남은 시간 막대와 카운트다운이 있는 향상된 버프 바 |
| 창 크기 | 기본 최소 크기와 공용 창 설정 | 최소화 기준 제거, 캐릭터별 창 크기 저장 |
| 폰트 | 기본 비트맵 폰트 중심 | 툴팁과 모던 UI 일부에 TTF 폰트 지원 |
| 장시간 플레이 | 수동 복구 | 프리징 감지, 재접속, 웹훅 알림 |
| 기타 | 기본 캐시/렌더러 | `flushanim`, 렌더러 및 언어 선택, 원본 CUO 버그 수정 반영 |

자세한 사용법은 [기능 둘러보기](features.md)에서 항목별로 확인하세요.
