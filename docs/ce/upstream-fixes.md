# 원본 CUO 패치 반영

GGO CE는 원본 ClassicUO 기반이므로 원본 수정 사항도 주기적으로 병합됩니다.

## 예시

GGO CE 1.6.0.3 반영 항목:

| 영역 | 내용 |
| --- | --- |
| 커스텀 하우징 | 커스텀 단계에서 발생할 수 있던 일부 버그 수정 |
| 마스터리 스펠북 | 현재 활성화된 마스터리 어빌리티가 책 우측 페이지에 정상 표시 |
| 거래창 | 골드/플래티넘 합산 잔액 기준으로 제안 한도 계산 |
| 책 | 페이지 줄 수와 글자색을 원본 클라이언트와 더 일치하도록 개선 |
| 안정성 | 맵 청크 관련 크래시와 렌더 오류 수정 |

## 제보 포인트

- 원본 CUO에서는 고쳐진 문제가 GGO CE에서 재현되는 경우
- 특정 원본 CUO 버전의 수정이 GGO CE에 아직 들어왔는지 확인이 필요한 경우
- 원본과 GGO CE의 동작 차이가 의도된 것인지 애매한 경우
