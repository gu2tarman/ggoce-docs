# 향상된 버프 바와 모던 UI

버프 시간 표시와 GGO 전용 UI 프레임을 묶은 기능입니다.

## 향상된 버프 바

![image.png](/ggoce-docs/assets/images/image-14.png)

켜는 위치:

`Options → GGO Custom → 향상된 버프 바`

- 버프별 남은 시간 막대 (최상단 한줄은 지속시간이 없거나 알 수 없는 버프를 아이콘으로 표시)
- 카운트다운 표시
- 캐릭터별 설정 저장
- 바 색상 조정
- HUD 스탯바와 접합

![image.png](/ggoce-docs/assets/images/image-15.png)

## 모던 UI

![image.png](/ggoce-docs/assets/images/image-16.png)

켜는 위치:

`Options → GGO Custom → GGO 모던 UI 사용`

기본값은 켜짐입니다.

적용 영역:

- 오토루팅 요약 검프
- 오토루팅 상세 검프
- 페이퍼돌 하단 이름표
- 향상된 버프 바 고정 기준 바
- 그리드 뷰 프레임과 일부 버튼
