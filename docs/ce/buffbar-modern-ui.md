# 향상된 버프 바와 모던 UI

버프 시간 표시와 GGO 전용 UI 프레임을 묶은 기능입니다.

## 향상된 버프 바

켜는 위치:

`Options → GGO Custom → 향상된 버프 바`

- 버프별 남은 시간 막대
- 카운트다운 표시
- 캐릭터별 설정 저장
- 바 색상 조정
- HUD 스탯바와 접합

## 모던 UI

켜는 위치:

`Options → GGO Custom → GGO 모던 UI 사용`

기본값은 켜짐입니다.

적용 영역:

- 오토루팅 요약 검프
- 오토루팅 상세 검프
- 페이퍼돌 하단 이름표
- 향상된 버프 바 고정 기준 바
- 그리드 뷰 프레임과 일부 버튼
