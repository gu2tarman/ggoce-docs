# 향상된 버프 바와 모던 UI

GGO CE는 버프 관리와 주요 GGO 검프의 시인성을 높이기 위해 향상된 버프 바와 모던 UI를 제공합니다.

## 향상된 버프 바

### 원본 CUO와 차이

원본 버프 바는 활성 버프 확인에 초점이 있습니다. GGO CE의 향상된 버프 바는 남은 시간 막대와 카운트다운을 함께 표시해 전투 중 갱신 타이밍을 더 쉽게 볼 수 있습니다.

### 켜는 위치

`Options → GGO Custom → 향상된 버프 바`

### 주요 기능

- 버프별 남은 시간 막대
- 카운트다운 표시
- 캐릭터별 설정 저장
- 바 색상 조정
- HUD 스탯바와 접합

## 모던 UI

### 적용 영역

- 오토루팅 요약 검프
- 오토루팅 상세 검프
- 페이퍼돌 하단 이름표
- 향상된 버프 바 고정 기준 바
- 그리드 뷰 프레임과 일부 버튼

### 켜는 위치

`Options → GGO Custom → GGO 모던 UI 사용`

기본값은 켜짐입니다. 기존 UI가 더 편한 경우 옵션에서 끌 수 있습니다.

## 설명 포인트

모던 UI는 기능 로직을 바꾸기보다 정보가 겹치거나 읽기 어려운 부분을 정리하는 목적입니다. 예를 들어 오토루팅 점수나 최근 획득 아이템은 원본 스타일보다 작은 공간에 압축되어 표시됩니다.
