# 그리드 뷰 확장 지원

가방, 상자, 백팩 같은 일반 컨테이너를 격자형 창으로 보는 기능입니다. 시체 그리드는 루팅용, 일반 컨테이너 그리드는 실제 아이템 조작용으로 동작이 조금 다릅니다. 업데이트를 통해 최종적으로는 일반 컨테이너는 끄고 그리드 뷰 만으로도 충분히 게임 플레이가 가능하도록 만드는 것이 목표입니다.

## 켜는 위치

1. `Options → Gumps & Context`에서 Grid Loot를 `Grid Only` 또는 `Both`로 설정합니다.
2. `Options → GGO Grid View`에서 일반 컨테이너 그리드 옵션을 켭니다.

## 조작


| 위치 | 조작 | 동작 |
| ------- | -------------------- | ---------------------------------------------------------- |
| 일반 컨테이너 | 클릭/더블클릭/드래그/타겟 커서 클릭 | 실제 아이템 조작과 동일 |
| 일반 컨테이너 | 컨트롤 + 클릭 / 컨트롤 + 드래그 | 아이템 즐겨찾기 / 즐겨찾기 순서 변경 (드래그 후 아이템 위에 드랍 = 해당 아이템 앞으로 순서 이동) |
| 일반 컨테이너 | 알트 + 클릭 / 알트 + 더블 클릭 | 아이템 다중 선택 / 같은 종류 아이템 모두 선택 |
| 일반 컨테이너 | 쉬프트 + 클릭 | 아이템 즉시 꺼내기 |
| 시체 그리드 | 클릭 / Shift+클릭/더블클릭 | 루팅 / 실제 아이템 조작 |


## 편의 기능

- 검색: OPL 내용 기준으로 필터링합니다.
- 스크롤: 아이템이 많으면 휠 또는 스크롤바로 이동합니다.
- 슬롯 크기: `50~80` 범위에서 조절합니다.
- 위치 저장: 일반 컨테이너는 컨테이너별, 시체는 공용 크기로 저장됩니다.
- 접합: 일반 컨테이너 그리드끼리 붙여 함께 이동할 수 있습니다.
- 즐겨찾기: Ctrl+클릭으로 지정하고, Ctrl+드래그로 순서를 바꿉니다.
- 최소화: 제목 영역을 더블클릭하면 작은 아이콘으로 접습니다.

## 주의 사항

- Grid Loot 기본 옵션이 `None`이면 작동하지 않습니다.
- 옵션을 바꾼 뒤 이미 열려 있는 컨테이너는 다시 열어야 새 설정이 반영될 수 있습니다.
- 시체 그리드는 접합 대상에서 제외됩니다.
