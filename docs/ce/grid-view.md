# 그리드 뷰 확장

GGO CE의 그리드 뷰는 원본 CUO의 Grid Loot 기능을 일반 컨테이너까지 확장한 기능입니다. 시체뿐 아니라 가방, 상자, 백팩 같은 일반 컨테이너도 격자형 목록으로 볼 수 있습니다.

## 원본 CUO와 차이

| 항목 | 원본 CUO | GGO CE |
| --- | --- | --- |
| 대상 | 주로 시체 Grid Loot | 시체 + 일반 컨테이너 |
| 조작 | 루팅 중심 | 일반 클릭/더블클릭/드래그/타겟 지정 지원 |
| 표시 | 기본 그리드 | 모던 프레임, 점수/분류 표시, 슬롯 크기 조절 |
| 관리 | 개별 창 위치 한계 | 크기/위치 저장, 접합, 즐겨찾기, 검색 |

## 켜는 위치

1. `Options → Gumps & Context`에서 Grid Loot를 `Grid Only` 또는 `Both`로 설정합니다.
2. `Options → GGO Grid View`에서 일반 컨테이너 그리드 뷰와 세부 옵션을 조절합니다.

## 주요 기능

### 일반 컨테이너 그리드

가방이나 상자를 열 때 기존 컨테이너 검프 대신 그리드 뷰로 볼 수 있습니다. 아이템이 많은 상자를 훑거나, 점수 표시를 보며 장비를 비교할 때 유용합니다.

### 일반 조작 지원

일반 컨테이너 그리드에서는 클릭과 더블클릭이 실제 아이템 조작처럼 동작합니다. 책, 상자, 사용 아이템도 그리드 안에서 바로 다룰 수 있고, 타겟 커서가 떠 있을 때는 그리드 아이템을 타겟 대상으로 지정할 수 있습니다.

### 시체 그리드 조작

시체 그리드는 루팅 보조 목적이므로 일반 컨테이너와 다르게 동작합니다.

| 조작 | 동작 |
| --- | --- |
| 일반 클릭 | 루팅 |
| Shift+클릭 | 실제 아이템 상호작용 |
| Shift+더블클릭 | 실제 아이템 더블클릭 |

### 스크롤과 슬롯 크기

아이템이 많으면 마우스 휠이나 우측 스크롤바로 이동합니다. 슬롯 크기는 `50~80` 범위에서 조절할 수 있고, 아이템 아이콘도 함께 커집니다.

### 크기와 위치 저장

일반 컨테이너는 컨테이너별로 위치와 크기를 기억합니다. 시체 그리드는 공용 크기를 사용합니다. 백팩이나 자주 쓰는 창고를 고정된 위치에 배치해두면 다음 접속에서도 비슷한 흐름으로 이어갈 수 있습니다.

### 접합

일반 컨테이너 그리드끼리 붙여서 함께 움직일 수 있습니다. 재접속 후 컨테이너가 다시 열리면 접합 상태도 복원됩니다. 시체 그리드는 접합 대상에서 제외됩니다.

### 즐겨찾기

Ctrl+클릭으로 즐겨찾기를 지정할 수 있습니다. Ctrl을 누른 상태에서 즐겨찾기 아이템을 드래그하면 순서를 조정할 수 있습니다.

### 검색과 필터

그리드 창 상단 검색창에 텍스트를 입력하면 OPL 내용 기준으로 필터링합니다. 특정 옵션, 이름, 재료를 빠르게 찾을 때 사용합니다.

### 최소화

그리드 창 제목 영역을 더블클릭하면 작은 아이콘 형태로 최소화됩니다. 위치를 기억하므로 창이 많은 상황에서 임시로 정리하기 좋습니다.

## 주의 사항

- Grid Loot 기본 옵션이 `None`이면 GGO 그리드 확장도 작동하지 않습니다.
- 옵션을 바꾼 뒤 이미 열려 있는 컨테이너는 다시 열어야 새 설정이 반영될 수 있습니다.
- 시체 그리드와 일반 컨테이너 그리드는 목적이 달라 조작 방식이 일부 다릅니다.
