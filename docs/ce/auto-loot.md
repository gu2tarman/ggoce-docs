# 오토루팅

GGO CE 오토루팅은 시체 안의 아이템 속성을 읽고, 점수와 규칙에 따라 필요한 아이템을 자동으로 줍거나 사용자가 확인해야 할 장비를 강조하는 기능입니다.

## 원본 CUO와 차이

원본 CUO는 기본 루팅 UI와 Grid Loot를 제공합니다. GGO CE는 여기에 아이템 속성 분석, 점수 계산, 커스텀 목록, 루팅 기록, 장비 추천 표시를 추가했습니다.

## 켜는 위치

`Options → GGO Auto Loot`

## 기본 흐름

1. 주변 시체를 자동으로 엽니다.
2. 아이템 OPL을 읽습니다.
3. 장비, 재료, 골드, 보물지도, 커스텀 목록을 판정합니다.
4. 조건에 맞는 아이템을 지정한 가방으로 이동합니다.
5. 처리한 시체를 강조하거나 닫아 중복 처리를 줄입니다.
6. 고가치 획득은 요약 검프와 기록에 남깁니다.

## 주요 옵션

| 옵션 | 설명 |
| --- | --- |
| 오토루팅 활성화 | 전체 기능을 켜고 끕니다. |
| 점수 기준 | 장비를 자동 루팅할 최소 점수입니다. 기본값은 `550`입니다. |
| 장비 가방 | 장비와 주요 루팅 대상이 들어갈 가방입니다. |
| 자원 가방 | 골드, 임뷰 재료 등 자원성 아이템이 들어갈 가방입니다. |
| 완료 시체 강조 | 처리 완료한 시체의 색상을 바꿔 중복 확인을 줄입니다. |
| 골드 | 골드 루팅 여부입니다. |
| 고레벨 보물지도 | 6레벨 이상 보물지도 루팅 여부입니다. |
| 임뷰 재료 | 임뷰 재료 루팅 여부입니다. |
| 운셋 Base | 운 150 이상, 상급 유물 이상 등 운 장비 후보를 봅니다. |
| 지원템 | 운 150, FC1/FCR3 같은 보조 가치 아이템을 봅니다. |
| 커스텀 아이템 | 사용자가 등록한 그래픽/이름 기준 아이템을 줍습니다. |

## 장비 판정

### 방어구와 방패

스탯과 주요 옵션을 합산해 기준 점수 이상이면 후보로 표시합니다. 저항과 운은 등급 표시에는 반영되지만, 저항만 높은 잡템이 무조건 통과하지 않도록 분리되어 있습니다.

### 무기

무기는 일반 점수보다 흡수 옵션을 중심으로 봅니다. 기본 기준은 체력 흡수와 마나 흡수가 모두 `30%` 이상인 경우입니다.

### 추천 표시

무기, 방어구, 방패 후보는 자동으로 줍기보다 시체 그리드에서 눈에 띄게 보여 직접 확인하도록 돕습니다. 전설 아티팩트처럼 명확한 대상은 판정과 무관하게 자동 루팅 대상이 될 수 있습니다.

## 커스텀 목록

커스텀 목록은 특정 아이템을 직접 등록해 루팅하는 기능입니다.

- 그래픽 기준으로 같은 베이스 아이템을 루팅할 수 있습니다.
- 이름 일치 옵션을 켜면 같은 베이스라도 지정한 이름과 일치할 때만 줍습니다.
- 흔한 베이스를 공유하는 특정 유니크 아이템을 선별할 때 유용합니다.

## 요약 검프와 기록

상단 메뉴바의 `Auto Loot` 버튼으로 오토루팅 요약 검프를 열 수 있습니다. 요약 검프에서는 오토루팅과 시체 자동 열기를 빠르게 켜고 끌 수 있고, 최근 획득 기록과 최고 점수를 확인할 수 있습니다.

## 시간 조정

네트워크나 서버 반응이 느릴 때는 다음 값을 조정할 수 있습니다.

| 항목 | 기본값 | 설명 |
| --- | --- | --- |
| 자동열기 간격 | 500ms | 다음 시체 자동 열기까지 최소 간격 |
| 오픈 보류 | 1500ms | 검프가 열리지 않았을 때 재시도 전 대기 |
| 액션 간격 | 500ms | 아이템 이동 액션 사이 간격 |

## 제보 포인트

오토루팅 제보에는 아이템 속성 스크린샷이 가장 중요합니다.

- 좋은 아이템을 놓친 경우
- 가치 없는 아이템을 주운 경우
- 커스텀 이름 일치가 예상과 다르게 동작한 경우
- 시체가 열리지 않거나 처리 완료 표시가 이상한 경우
