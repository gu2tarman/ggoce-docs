# 오토루팅

시체 아이템을 읽어 점수와 규칙에 맞는 아이템을 줍는 기능입니다. 무기/방어구 후보는 자동으로 줍기보다 시체 그리드에서 확인하기 쉽게 표시합니다.

## 켜는 위치

`Options → GGO Auto Loot`

## 처리 흐름

1. 주변 시체를 자동으로 엽니다.
2. 아이템 OPL을 읽습니다.
3. 점수/규칙/커스텀 목록을 판정합니다.
4. 조건에 맞는 아이템을 지정 가방으로 옮깁니다.
5. 처리한 시체를 표시하고, 고가치 획득은 기록에 남깁니다.

## 옵션

| 옵션 | 설명 |
| --- | --- |
| 오토루팅 활성화 | 전체 기능 on/off |
| 점수 기준 | 장비 자동 루팅 기준. 기본값 `550` |
| 장비 가방 | 장비와 주요 루팅 대상 이동 위치 |
| 자원 가방 | 골드, 임뷰 재료 등 자원성 아이템 이동 위치 |
| 완료 시체 강조 | 처리한 시체 색상 표시 |
| 골드 / 보물지도 / 임뷰 재료 | 각 분류별 루팅 여부 |
| 운셋 Base | 운 150 이상 장비 후보 |
| 지원템 | 운 150, FC1/FCR3 등 보조 가치 아이템 |
| 커스텀 아이템 | 사용자가 등록한 아이템 |

## 장비 판정

| 종류 | 기준 |
| --- | --- |
| 방어구/방패 | 스탯과 주요 옵션 점수 |
| 무기 | 체력 흡수 + 마나 흡수. 기본 기준 `30%` |
| 전설 아티팩트 | 판정과 무관하게 자동 루팅 대상 |

## 커스텀 목록

특정 아이템을 직접 등록해 루팅합니다. 이름 일치를 켜면 같은 베이스 아이템이라도 지정한 이름과 맞을 때만 줍습니다.

## 요약 검프와 기록

상단 메뉴바의 `Auto Loot` 버튼으로 요약 검프를 엽니다. 여기서 오토루팅/시체 자동 열기를 빠르게 켜고 끄고, 최근 획득 기록을 확인할 수 있습니다.

## 시간 조정

| 항목 | 기본값 | 설명 |
| --- | --- | --- |
| 자동열기 간격 | 500ms | 다음 시체 자동 열기까지 최소 간격 |
| 오픈 보류 | 1500ms | 검프가 열리지 않았을 때 재시도 전 대기 |
| 액션 간격 | 500ms | 아이템 이동 액션 사이 간격 |

## 제보 포인트

아이템 판정 제보는 아이템 속성 스크린샷이 있어야 확인하기 쉽습니다.
