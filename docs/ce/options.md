# 옵션 위치

GGO CE의 주요 기능은 게임 내 옵션 창의 GGO 관련 탭에서 설정합니다.

## 자주 찾는 옵션

| 항목 | 위치 |
| --- | --- |
| GGO 모던 UI | Options → GGO Custom |
| HUD 스탯바 | Options → GGO Custom |
| 향상된 버프 바 | Options → GGO Custom |
| 그리드 뷰 설정 | Options → GGO Grid View |
| 오토루팅 | Options → GGO Auto Loot |
| 오토루팅 가방/자원 가방 | Options → GGO Auto Loot |
| 무기/방어구 판정 | Options → GGO Auto Loot |
| 루팅 간격 | Options → GGO Auto Loot |
| 서버 프리징 감지 | Options → GGO Custom |
| 회복 헬퍼 | Options → GGO Custom |
| GGO 체력바 스킨 | Options → GGO Custom |
| 파티/펫 체력바 자동 숨김 | Options → GGO Custom |
| 발밑 헬스바 상시 표시 | Options → GGO Custom |
| 스탯 가드 | Options → GGO Custom |
| 그래픽 렌더러 | Options → GGO Custom |

!!! note
    탭 이름과 항목 위치는 버전에 따라 조금씩 정리될 수 있습니다. 문서와 실제 화면이 다르면 스크린샷과 함께 제보해 주세요.
