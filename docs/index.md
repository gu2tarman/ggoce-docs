# GGO CE Guide

GGO CE와 GGO Custom Launcher, 그리고 GGO 스크립트를 처음 쓰는 유저를 위한 공식 가이드입니다.

## 빠른 시작

1. [설치와 업데이트](install/launcher.md)에서 기존 PPT 설정 가이드를 확인합니다.
2. [첫 실행 체크리스트](install/first-run.md)대로 기본 경로와 프로필을 점검합니다.
3. 자주 쓰는 기능은 [GGO CE 기능 둘러보기](ce/features.md)와 [스크립트 전체 목록](scripts/overview.md)에서 찾습니다.
4. 문제가 생기면 [문제 해결](install/troubleshooting.md)을 먼저 확인한 뒤 [피드백 보내기](support/feedback.md)로 제보합니다.

## 문서 운영 원칙

- 런처 공지는 짧게, 자세한 설명은 이 문서에 정리합니다.
- 반복되는 질문은 FAQ로 승격합니다.
- 버그 제보는 재현 단계, 스크린샷, 사용 버전을 함께 받습니다.
- 기능 제안은 토론 후 실제 작업 이슈로 옮깁니다.
