# GGO_쓰로잉봇

가고일 팔라딘 원거리 전사용 파티봇입니다. 리더를 따라다니며 원거리에서 적을 공격하고, 적이 가까워지면 거리를 벌립니다.

## 준비물

- 룬북
- 백팩 안의 쓰로잉 무기
- Razor Enhanced Dress 에이전트
- Razor Enhanced Vendor Sell 에이전트
- Razor Enhanced Organizer 에이전트

## 주요 기능

- 리더 추적
- 원거리 전투 유지
- 접근한 적을 상대로 카이팅
- Consecrate Weapon, Divine Fury 유지
- 뱀파이어폼과 비행 활성화 보조

## 주요 명령어

| 명령어 | 동작 |
| --- | --- |
| `!max` | 원거리 최대 유지 모드 |
| `!선공` | 범위 내 적 선공 모드 |
| `!챔프 [캐릭명]` | 현재 위치를 기준으로 챔프 모드 |
| `!정상화` | 일반 모드 복귀 |
| `!아머` | Armor Ignore 모드 토글 |
| `!뱀폼` | 뱀파이어폼 시전 |
| `!날아` | 비행 활성화 |
| `!s[번호]` | 룬북 위치로 Sacred Journey 이동 |
