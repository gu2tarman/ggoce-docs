# 스크립트 전체 목록

GGO 스크립트는 Razor Enhanced에서 실행하는 자동화와 보조 도구 모음입니다.

## 설정 파일 위치

```text
%APPDATA%\GGO_Settings\[스크립트명]\script_settings.json
%APPDATA%\GGO_Settings\[스크립트명]\[캐릭명].json
```

## 주요 스크립트

| 스크립트 | 용도 |
| --- | --- |
| GGO_오토루팅 | 시체 아이템을 점수 기준으로 자동 루팅합니다. |
| GGO_바울바드봇인핸스 | 바드 파티봇입니다. |
| GGO_쓰로잉봇 | 가고일 쓰로잉 파티봇입니다. |
| GGO_업데이트매니저 | 스크립트 최신 버전을 확인하고 업데이트합니다. |
| GGO_바울마이닝인핸스 | 룬북 기반 자동 채광입니다. |
| GGO_바울럼버인핸스 | 룬북 기반 자동 벌목입니다. |
| GGO_바울리콜피싱인핸스 | 룬북 기반 바다 낚시입니다. |
| GGO_바울SOS파머 | SOS 상자 파밍을 보조합니다. |
| GGO_레지듀제작기 | 단검 제작과 임뷰 해체로 residue를 생성합니다. |
| GGO_쓰포모으기 | 학살자 장궁 제작으로 trash point를 쌓습니다. |

## 업데이트

스크립트는 [업데이트매니저](update-manager.md)를 통해 최신 버전으로 갱신하는 것을 권장합니다.
