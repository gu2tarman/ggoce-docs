# GGO_바울바드봇인핸스

바드 캐릭터 전용 파티봇입니다. 리더를 따라다니며 디스코던스와 프로보케이션으로 적을 약화시키고, 파티원과 펫 회복을 보조합니다.

## 준비물

- 백팩 안의 악기
- 룬북
- 펫 먹이
- Razor Enhanced Dress 에이전트
- Razor Enhanced Vendor Sell 에이전트
- Razor Enhanced Organizer 에이전트

## 기본 흐름

1. 리더를 지정합니다.
2. 룬북을 지정합니다.
3. 파티원과 펫을 지정합니다.
4. 시작 버튼으로 봇을 작동합니다.

## 주요 명령어

| 명령어 | 동작 |
| --- | --- |
| `!정지` / `!재개` | 봇을 일시정지하거나 재개합니다. |
| `!팔로우` | 공격을 중단하고 리더만 추적합니다. |
| `!정상화` | 일반 전투 모드로 돌아갑니다. |
| `!디코` / `!프로보` | 바드 스킬 모드를 전환합니다. |
| `!밥줘` | 펫에게 먹이를 줍니다. |
| `!sell` | 파워스크롤 판매를 수행합니다. |
| `r[번호]` / `g[번호]` | 룬북 위치로 리콜 또는 게이트 이동합니다. |
