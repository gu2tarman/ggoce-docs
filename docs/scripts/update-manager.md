# GGO_업데이트매니저

GGO 스크립트 최신 버전을 확인하고 한 번에 업데이트하는 도구입니다.

## 사용 방법

1. Razor Enhanced 스크립트 탭에서 `GGO_업데이트매니저.py`를 실행합니다.
2. 목록에서 현재 버전과 최신 버전을 확인합니다.
3. 필요한 스크립트만 개별 업데이트하거나 전체 업데이트를 실행합니다.
4. 업데이트 후 Razor Enhanced 스크립트 목록을 새로고침합니다.

## 주의 사항

- 실행 중인 스크립트 파일을 업데이트하면 바로 반영되지 않을 수 있습니다.
- 업데이트 후에는 관련 스크립트를 다시 시작하는 것을 권장합니다.
- 네트워크 오류가 반복되면 잠시 후 다시 시도하거나 문의 채널로 제보합니다.
