# GGO_오토루팅

주변 시체를 검사해 점수 기준을 넘는 장비, 골드, 임뷰 재료, 커스텀 아이템을 지정 가방으로 루팅합니다.

## 동작 방식

1. 반경 내 시체를 감지합니다.
2. 아이템 속성을 읽습니다.
3. 가중치 점수를 계산합니다.
4. 기준을 넘는 아이템을 지정 가방으로 이동합니다.
5. 최근 획득 아이템과 점수를 대시보드에 표시합니다.

## 루팅 모드

| 모드 | 내용 |
| --- | --- |
| 집중 루팅 | 기준 점수 이상 장비와 전설 아티팩트를 줍습니다. |
| 지원템 루팅 | 운, FC/FCR 등 보조 가치가 있는 아이템도 포함합니다. |
| 임뷰 루팅 | 낮은 무게의 매직 아이템을 임뷰 재료 후보로 포함합니다. |
| 극한 노가다 | 더 넓은 범위의 매직 아이템을 수집합니다. |

## 주요 버튼

| 버튼 | 설명 |
| --- | --- |
| Perm | 아이템을 영구 커스텀 목록에 추가합니다. |
| Sess | 이번 세션에만 커스텀 목록에 추가합니다. |
| Det | 최근 획득 장비의 상세 판정을 봅니다. |
| Loot | 선택한 상자나 시체를 즉시 수동 루팅합니다. |

## 제보 팁

좋은 아이템을 놓치거나 이상한 아이템을 줍는 경우, 아이템 스크린샷과 속성 정보를 함께 보내 주세요. 점수표 개선에 가장 도움이 됩니다.
