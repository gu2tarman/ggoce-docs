# 설치와 업데이트

최초 설치는 PPT 설정 가이드를 기준으로 진행합니다.

[설정 가이드 열기](https://docs.google.com/presentation/d/1iA6vzvoBJPdn_FnCCv4HwOVrhrgPSv4OAhWop-rcqAo/present?usp=sharing){ .md-button .md-button--primary }

## 순서

1. PPT 설정 가이드대로 기본 설치와 접속 설정을 완료합니다.
2. GGO Custom Launcher를 실행합니다.
3. GGO CE 설치 폴더를 선택합니다.
4. 런처에서 업데이트를 진행합니다.
5. `PLAY`로 실행되는 클라이언트가 의도한 폴더인지 확인합니다.

## 업데이트가 보는 버전

런처는 `ClassicUO.exe`가 아니라 `cuo.dll`의 파일 버전을 기준으로 GGO CE 버전을 판단합니다. 업데이트 대상도 manifest에 들어 있는 파일만 검사합니다.

!!! warning "원본 ClassicUO 폴더 주의"
    GGO CE는 원본 ClassicUO와 별도 폴더에 설치하는 것을 권장합니다.
