# 문제 해결

## 업데이트가 실패합니다

1. 런처를 완전히 종료한 뒤 다시 실행합니다.
2. 백신이나 보안 프로그램이 다운로드 파일을 격리했는지 확인합니다.
3. 설치 폴더가 쓰기 가능한 위치인지 확인합니다.
4. 같은 문제가 반복되면 로그와 스크린샷을 첨부해 제보합니다.

## 실행 직후 꺼집니다

- 원본 ClassicUO 파일과 GGO CE 파일이 섞였는지 확인합니다.
- `cuo.dll`만 수동으로 바꿔 끼운 상태라면 새 폴더에 다시 설치하는 편이 안전합니다.
- 런처와 클라이언트를 모두 종료한 뒤 업데이트를 다시 진행합니다.

## UI 위치나 창 크기가 이상합니다

- 캐릭터별 프로필 설정이 꼬였을 수 있습니다.
- 옵션을 다시 저장한 뒤 클라이언트를 정상 종료합니다.
- 여러 클라이언트를 동시에 켠 상태에서 같은 프로필을 수정하지 않는 것이 좋습니다.

## 제보할 때 필요한 정보

- 런처 버전
- GGO CE 버전
- Windows 버전
- 설치 경로
- 재현 순서
- 오류 화면 스크린샷
