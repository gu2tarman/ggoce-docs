# 문제 해결

## 업데이트가 실패합니다

1. 런처를 완전히 종료한 뒤 다시 실행합니다.
2. 설치 폴더가 쓰기 가능한 위치인지 확인합니다.
3. 백신이나 보안 프로그램이 파일을 막았는지 확인합니다.
4. 반복되면 오류 화면과 설치 경로를 함께 제보합니다.

## 실행 직후 꺼집니다

- 원본 ClassicUO 폴더와 GGO CE 폴더가 섞였는지 확인합니다.
- `cuo.dll`만 수동 교체한 상태라면 새 폴더에 다시 설치하세요.
- 런처와 클라이언트를 모두 종료한 뒤 업데이트를 다시 진행합니다.

## UI 위치나 창 크기가 이상합니다

옵션을 다시 저장한 뒤 클라이언트를 정상 종료합니다. 여러 클라이언트에서 같은 프로필을 동시에 수정하면 위치 저장이 꼬일 수 있습니다.

## 제보할 때 필요한 정보

- 런처 버전
- GGO CE 버전
- Windows 버전
- 설치 경로
- 재현 순서
- 오류 화면 스크린샷
