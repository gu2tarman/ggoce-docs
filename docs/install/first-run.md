# 첫 실행 체크리스트

처음 실행할 때는 아래 항목을 한 번만 확인하면 이후 문의와 오류를 크게 줄일 수 있습니다.

!!! note "최초 설치 안내"
    설치 순서와 기본 설정은 [설정 가이드 PPT](https://docs.google.com/presentation/d/1iA6vzvoBJPdn_FnCCv4HwOVrhrgPSv4OAhWop-rcqAo/present?usp=sharing)를 먼저 확인하세요. 이 문서는 PPT를 본 뒤 마지막 점검용으로 사용하면 됩니다.

## 런처

- GGO CE 설치 폴더가 원본 ClassicUO 폴더와 분리되어 있는지 확인합니다.
- 업데이트가 끝난 뒤 `PLAY`로 실행되는 클라이언트가 의도한 폴더인지 확인합니다.
- 멀티 로그인 사용 시 계정명과 캐릭터명을 정확히 입력합니다.

## 클라이언트

- 옵션의 GGO 관련 탭에서 필요한 기능만 켭니다.
- 창 크기와 주요 UI 위치를 한 번 저장합니다.
- 자동 복구, 웹훅, 루팅 같은 기능은 한 번에 전부 켜지 말고 하나씩 확인합니다.

## 스크립트

- Razor Enhanced 스크립트 폴더에 최신 파일이 들어 있는지 확인합니다.
- 스크립트별 첫 실행 타겟팅을 천천히 완료합니다.
- 설정 파일 위치를 기억해 둡니다.

```text
%APPDATA%\GGO_Settings\[스크립트명]\script_settings.json
%APPDATA%\GGO_Settings\[스크립트명]\[캐릭명].json
```
