# 첫 실행 체크리스트

!!! note "최초 설치 안내"
    설치 순서는 [설정 가이드 PPT](https://docs.google.com/presentation/d/1iA6vzvoBJPdn_FnCCv4HwOVrhrgPSv4OAhWop-rcqAo/present?usp=sharing)를 먼저 확인하세요.

## 런처

- GGO CE 설치 폴더가 원본 ClassicUO 폴더와 분리되어 있는지 확인합니다.
- `PLAY`가 어느 폴더의 클라이언트를 실행하는지 확인합니다.
- 멀티 로그인 사용 시 계정명과 캐릭터명을 확인합니다.

## 클라이언트

- 창 크기와 주요 UI 위치를 한 번 저장합니다.
- GGO 기능은 필요한 것부터 하나씩 켭니다.
- 자동 복구, 웹훅, 오토루팅은 먼저 낮은 위험 환경에서 테스트합니다.

## 스크립트

스크립트 가이드는 별도 준비 중입니다. 설정 파일은 보통 아래 경로에 저장됩니다.

```text
%APPDATA%\GGO_Settings\[스크립트명]\script_settings.json
%APPDATA%\GGO_Settings\[스크립트명]\[캐릭명].json
```
